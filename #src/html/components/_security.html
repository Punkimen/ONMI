Security

Conception

The modular layout of the ONMI structural security's technical implementation is its defining characteristic.

Each active element in the overall structure performs its function strictly, and each module is an autonomous program with its own source code and personal security concept, indirectly affecting the overall security picture.

A device like this is fully consistent with the philosophy of blockchain platforms.

Software sustainability

All computational processes underlying the game mechanics are carried out in external environments to avoid overloading the data storage system within the blockchain platform.

Additional computational power within the blockchain platform is unused, as each transaction transmits to the block only necessary information about the current state of the game to achieve maximum transparency of mathematical processes.

ONMI is a living structure. We can only imagine what the future of ONMI will be, but we know for sure that AI technology will play its part: based on the patterns of user behavior, it will develop its scenario for the future.

However, we believe that AI can never replace community - and even more so the lively emotions of communicating with like-minded people.

Therefore, our priority is to develop ONMI's social features.

In its philosophy, onmi is a nexus gen-app, in which the user's interaction with content and its author is rediscovered through story and gamification. Financial tools, chat rooms, marketplace - the inseparable link between the entertainment content and the service app that will be part of the new society.
